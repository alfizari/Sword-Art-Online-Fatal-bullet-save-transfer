# Sword-Art-Online-Fatal-bullet-save-transfer
Transfer your save file from PC to PS4 and PS4 to PC


For PS4 saves, you will first need to decrypt them using discord bots (HTOS server: https://discord.com/invite/htos, xcier server) or use garlic save to decrypt https://garlicsaves.com/

After decrypting and transfering PC save to PS4, you will need to encrypt your save again
